Método Científico

El proceso que se sigue para trabajar en ciencia tiene varias etapas:

  1. Observación e identificación del problema
  2. Hipótesis
  3. Experimentación y análisis de resultados
    • Si no confirma la hipótesis → volver al paso 2
    • Si confirma la hipótesis → paso 5
  4. Enunciar leyes y teorías
  5. Comunicación a la comunidad científica

Magnitud y Medida

Una magnitud es la propiedad de un cuerpo que puede medirse.

Ejemplos: longitud, masa, dureza, tiempo, capacidad, volumen…

¿Qué es medir?

Es comparar el tamaño de una magnitud con otra que tomamos de referencia. El resultado de una medida es un número y una unidad (m, cm, kg…)

Por acuerdo internacional en ciencia siempre se trabaja en el Sistema Internacional de Unidades (S.I).
Las magnitudes pueden ser: básicas/fundamentales y derivadas. Las derivadas pueden obtenerse a través de las básicas.

Notación Científica

Se usa para escribir cantidades muy grandes o muy pequeñas usando potencias de base 10:

Formato: m x 10^N

  • m: Coeficiente, 1 ≤ |m| < 10, puede ser decimal y positivo o negativo.
  • 10: Base de la potencia, siempre 10.
  • N: Exponente, entero, positivo o negativo. N+ → punto decimal a la derecha, N- → punto decimal a la izquierda.
